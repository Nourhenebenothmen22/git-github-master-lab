Git Cheat Sheet

Une fiche rapide, filtrable et animée pour retrouver la bonne commande Git au bon moment.

34 commandes13 catégories
34 commandes affichéesToutes les catégories
Configuration

git config --global user.name "Ada"

Essentiel

Définit le nom d'auteur utilisé dans vos commits.

Exemplegit config --global user.name "Ada Lovelace"
Configuration

git config --global user.email

Essentiel

Associe une adresse e-mail à votre identité Git.

Exemplegit config --global user.email "ada@example.com"
Dépôt local

git init

Essentiel

Initialise un nouveau dépôt Git dans le dossier courant.

Exemplegit init
Dépôt local

git clone <url>

Essentiel

Copie un dépôt distant et son historique complet.

Exemplegit clone https://github.com/user/projet.git
Informations

git status

Essentiel

Affiche l'état des fichiers et de la zone de staging.

Exemplegit status --short
Staging

git add <fichier>

Essentiel

Prépare un fichier pour le prochain commit.

Exemplegit add index.html
Staging

git add .

Important

Ajoute toutes les modifications du dossier courant.

Exemplegit add .
Staging

git restore --staged

Important

Retire un fichier du staging sans supprimer son contenu.

Exemplegit restore --staged index.html
Commit

git commit -m "message"

Essentiel

Enregistre les changements préparés dans l'historique Git.

Exemplegit commit -m "Add homepage"
Commit

git commit --amend

Important

Modifie le dernier commit local avant de le partager.

Exemplegit commit --amend -m "Fix homepage title"
Branches

git branch

Essentiel

Liste, crée ou supprime des branches locales.

Exemplegit branch feature/navbar
Branches

git switch -c <branche>

Essentiel

Crée une branche et bascule dessus immédiatement.

Exemplegit switch -c feature/design
Branches

git checkout

Important

Ancienne commande polyvalente pour changer de branche ou restaurer.

Exemplegit checkout main
Merge

git merge <branche>

Essentiel

Intègre une branche dans la branche courante.

Exemplegit merge feature/design
Merge

git merge --abort

Important

Annule un merge en cours lorsqu'un conflit devient trop risqué.

Exemplegit merge --abort
Remote GitHub

git remote -v

Essentiel

Affiche les dépôts distants liés au projet.

Exemplegit remote -v
Remote GitHub

git remote add origin

Essentiel

Connecte le dépôt local à un dépôt GitHub.

Exemplegit remote add origin https://github.com/user/projet.git
Pull / Push

git fetch

Important

Télécharge les références distantes sans les fusionner.

Exemplegit fetch --prune origin
Pull / Push

git pull --rebase

Essentiel

Récupère les changements distants avec un historique linéaire.

Exemplegit pull --rebase origin main
Pull / Push

git push -u origin main

Essentiel

Publie une branche et configure son suivi distant.

Exemplegit push -u origin main
Informations

git diff

Essentiel

Compare les modifications non encore validées.

Exemplegit diff --staged
Informations

git log --oneline

Essentiel

Affiche l'historique de façon compacte.

Exemplegit log --oneline --graph --all
Informations

git show HEAD

Important

Affiche le détail du dernier commit.

Exemplegit show HEAD
Informations

git blame <fichier>

Avancé

Indique quel commit a modifié chaque ligne.

Exemplegit blame src/app.js
Annulation

git restore <fichier>

Important

Annule les modifications locales d'un fichier.

Exemplegit restore index.html
Annulation

git reset --soft HEAD~1

Avancé

Défait le dernier commit tout en gardant les changements stagés.

Exemplegit reset --soft HEAD~1
Annulation

git revert <sha>

Important

Crée un commit inverse sans réécrire l'historique partagé.

Exemplegit revert a1b2c3d
Stash

git stash push -m "WIP"

Important

Met temporairement de côté un travail non terminé.

Exemplegit stash push -m "WIP formulaire"
Stash

git stash pop

Important

Réapplique le dernier stash et le retire de la pile.

Exemplegit stash pop
Tags

git tag -a v1.0.0

Important

Crée un tag annoté pour marquer une version stable.

Exemplegit tag -a v1.0.0 -m "Release 1.0.0"
Tags

git push origin v1.0.0

Important

Publie un tag vers GitHub.

Exemplegit push origin v1.0.0
Nettoyage

git clean -fd

Avancé

Supprime les fichiers non suivis du dépôt local.

Exemplegit clean -fd
Nettoyage

git gc

Avancé

Optimise la base d'objets Git du dépôt.

Exemplegit gc --prune=now
Branches

git cherry-pick <sha>

Avancé

Applique un commit précis sur la branche courante.

Exemplegit cherry-pick a1b2c3d
